O que é Parallax?

O parallax é um efeito visual em que a posição de um objeto parece mudar quando visto de diferentes ângulos. Na web, é comum usar o parallax para criar um efeito de profundidade em páginas com imagens de fundo.

Como criar Parallax?

Para criar o efeito de parallax, geralmente usamos CSS e JavaScript. No entanto, neste exemplo, vamos usar apenas CSS. O segredo é adicionar uma imagem de fundo e configurá-la com uma posição fixa e um tamanho grande o suficiente para cobrir toda a tela.
